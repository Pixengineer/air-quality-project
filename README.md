# 🌍 Air Quality Prediction & Monitoring System

## 📌 Overview
This project predicts the Air Quality Index (AQI) using machine learning techniques based on pollutants such as PM2.5, PM10, NO2, CO, SO2, and O3. It helps in analyzing pollution trends and provides insights for environmental monitoring and public awareness.

## 🚀 Live Demo
🔗 https://air-quality-project-3.streamlit.app/

## 🚀 Features
- Data analysis and visualization of air quality parameters 📊  
- Machine learning-based AQI prediction 🤖  
- Interactive graphs and trend analysis 📈  
- User-friendly interface using Streamlit 🌐  
- Scalable for real-time data integration  

## 🧠 Tech Stack

## 📊 Dataset
The dataset includes major air pollutants:
- PM2.5  
- PM10  
- NO2  
- CO  
- SO2  
- O3  

Data sources: CPCB (India) / Kaggle datasets  

## ⚙️ Installation & Setup
1. Clone the repository:
   git clone https://github.com/Pixengineer/air-quality-project.git  
   cd air-quality-project  

2. Install dependencies:
   pip install -r requirements.txt  

3. Run the app:
   streamlit run app.py  

## 📈 Model Details
- Input: Pollution parameters  
- Output: AQI value / category  
- Algorithms used: Linear Regression / Random Forest
